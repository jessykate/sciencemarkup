Microdata and RDFa
  • Explanation
  • Both standards are under development
  • Focusing on microdata for now, but would be happy for RDFa implementations too.
Citation
  • Self-certifying: each object's ID is a hash, computed from the primary fields of the research object, including the author's name and the object's content.
  • This hash is also used as the element id on the page, so that citation and recognition are literally built into the linking process.
